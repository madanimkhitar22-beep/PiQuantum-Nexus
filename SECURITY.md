# Security Policy

## Supported Versions
The current stable release of **PiQuantum Nexus** is supported with continuous updates, including security patches and protocol hardening.

| Version | Supported |
|----------|------------|
| v1.0.x   | ✅ Active |
| v0.x     | ❌ Deprecated |

## Reporting a Vulnerability
If you discover a security issue, please report it responsibly through the PiTrust Secure Disclosure System.

- **Email:** security@pitrust.io  
- **Response Time:** Within 48 hours  
- **PGP Key:** Available on request  

Please include a detailed description of the vulnerability, reproduction steps, and potential impact. Do not publicly disclose the issue before an official fix is released.

## Security Measures
- Quantum-resistant encryption (Hybrid AES-512 + Lattice cryptography)
- Enforced HTTPS and HSTS
- Continuous code auditing through PiOS Governance
- Multi-layer identity verification via PiTrust

## Responsible Disclosure
We appreciate ethical researchers and contributors helping us secure the ecosystem.
